# Documentation de base des Blueprints Unreal Engine

Ce fichier offre un point de départ pour documenter les principaux nœuds (nodes) Blueprints. Pour chaque nœud, ajoutez un titre, une courte description et une capture d'écran tirée de la vidéo [Unreal Engine Blueprints – Full Course](https://www.youtube.com/watch?v=KE_8OwmHY-A) au moment où le nœud est présenté.

> **Astuce :** Vous pouvez réaliser une capture d'écran en mettant la vidéo en pause puis en utilisant l'outil de capture de votre système.

---

## Exemple de structure

### 1. Event BeginPlay
**Description :** Point d'entrée exécuté une seule fois au démarrage de l'acteur ou du niveau.

![Event BeginPlay](images/event_begin_play.png)

### 2. Event Tick
**Description :** Fonction qui se déclenche à chaque frame; utile pour mettre à jour des comportements en continu.

![Event Tick](images/event_tick.png)

### 3. Branch
**Description :** Noeud de condition qui exécute un chemin différent selon qu'une condition soit vraie ou fausse.

![Branch](images/branch.png)

### 4. Print String
**Description :** Permet d'afficher rapidement un message à l'écran ou dans la console pour le débogage.

![Print String](images/print_string.png)
